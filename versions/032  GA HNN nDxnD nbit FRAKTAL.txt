### **1. Sistemos apžvalga**

Jūsų pateiktas kodas yra patobulinta ir labiau struktūrizuota ankstesnio projekto versija. Sistemoje įdiegta fraktalais pagrįsta vaizdo kompresija, kuri optimizuojama naudojant **genetinį algoritmą**. Naujojo kodo stipriosios pusės – aiškumas, moduliai ir pramoniniai inžineriniai sprendimai.

**Sistemos pagrindiniai komponentai:**

  * **`SystemConfig`**: Centralizuota konfigūracijos klasė, apibrėžianti tokius parametrus kaip įrenginio tipas, įvesties formos, VRAM ribos ir failų keliai. Tai yra gera praktika, leidžianti lengvai keisti sistemos nustatymus.
  * **`FractalProcessor`**: Vykdo fraktalinį kodavimą ir dekodavimą. Jame integruoti **CUDA branduoliai**, kurie atlieka pagrindinius skaičiavimus GPU.
  * **`GeneticOptimizer`**: Klasė, atsakinga už genetinio algoritmo logiką – populiacijos inicializavimą, vertinimą, kryžminimą ir mutacijas.
  * **CUDA branduoliai (`fractal_encode_kernel`, `fractal_decode_kernel`)**: Žemo lygio funkcijos, parašytos CUDA, kurios turėtų pagreitinti fraktalų skaičiavimus.
  * **Duomenų struktūros**: `dataclass` ir `Enum` naudojimas genomo parametrams aprašyti padidina kodo skaitomumą ir struktūrą.

-----

### **2. Kompresijos efektyvumo vertinimas**

Jūsų kodas naudoja naujovišką hibridinį kompresijos metodą, kuris derinamas su genetiniu algoritmu. Tačiau pramoniniu požiūriu pastebimi keli esminiai trūkumai, kurie mažina bendrą efektyvumą.

#### **Stipriosios pusės**

  * **Spartos optimizavimas**: Naudojami **CUDA branduoliai** leidžia atlikti pagrindinius skaičiavimus lygiagrečiai GPU, o tai drastiškai padidina apdorojimo greitį.
  * **Pridėtinė vertė (Cache)**: `FractalProcessor` klasėje įdiegtas **cache** mechanizmas, kuris saugo jau apskaičiuotas transformacijas. Tai yra puikus sprendimas, leidžiantis pagreitinti pasikartojančių duomenų kodavimą.
  * **Struktūrizavimas**: Duomenų, tokių kaip `FractalParams` ir `TrainingParams`, atskyrimas gerina kodo skaitomumą ir leidžia lengviau valdyti genomo struktūrą.
  * **Asinchroniškumas**: `asyncio` naudojimas populiacijos vertinimui teoriniu lygmeniu leis lygiagrečiai apdoroti kelis genomus, taip pagreitinant visą evoliucijos procesą.

#### **Silpnosios pusės ir tobulinimo sritys**

  * **Apskaičiavimų paprastumas**: Jūsų **CUDA branduoliai** yra per daug supaprastinti ir neatlieka tikrojo fraktalinio vaizdo kodavimo. `fractal_encode_kernel` funkcija grąžina fiksuotas reikšmes (0.5f, 0.1f), o ne realias transformacijas, kurios būtų apskaičiuojamos pagal įvesties duomenis. Toks supaprastinimas iš esmės paverčia kompresijos algoritmą netikru, todėl gaunamos PSNR reikšmės tėra atsitiktinio triukšmo arba artefaktų rezultatas.
  * **PSNR skaičiavimas**: `peak_signal_noise_ratio` funkcijoje naudojamas `data_range=1.0`, o tai gali būti netikslu, jei įvesties duomenys nėra normalizuoti intervale [0, 1]. Dėl neteisingų reikšmių gali būti gaunami klaidingi PSNR rezultatai, kas iškraipo genetinio algoritmo fitneso vertinimą.
  * **Optimizatoriaus trūkumai**: `GeneticOptimizer` naudoja tuščią `optim.Adam` optimizatorių ( ` optim.Adam([], ...)  `), o tai reiškia, kad jokie modelio svoriai netreniruojami. Kadangi fraktalinis modelis neturi treniruojamų parametrų, tai nėra klaida, bet kodo logika neatspindi tikrojo mašininio mokymosi. Tai rodo, kad sistema yra labiau **parametrų paieškos algoritmas** nei tinklo treniravimo sistema.
  * **VRAM valdymas**: Nors `torch.cuda.empty_cache()` yra gerai, `if genome['vram_usage'] > SystemConfig.VRAM_LIMIT_MB * 0.8` sąlyga gali ne visada veikti, nes VRAM naudojimas skaičiuojamas tik po kodavimo/dekodavimo, kai didžioji dalis atminties jau gali būti užpildyta. Kritiniais atvejais tai gali sukelti `OutOfMemoryError` dar iki šios sąlygos įvykdymo.

### **4. Išvados ir rekomendacijos**

Nors jūsų kodas yra gerai struktūrizuotas ir naudoja pažangius inžinerinius metodus (CUDA, asyncio, dataclasses), **tikrasis kompresijos algoritmas yra per daug supaprastintas**. Dėl to sistema iš esmės testuoja tik skirtingus parametrus su fiksuotu ir neefektyviu kodu.

**Rekomenduojami patobulinimai:**

1.  **Išplėskite CUDA branduolius**: Perrašykite `fractal_encode_kernel` taip, kad jis iš tikrųjų rastų **domeno ir diapazono blokus** bei apskaičiuotų afininės transformacijos koeficientus. Tai yra esminė fraktalinės kompresijos dalis.
2.  **Įveskite tikrąjį mokymą**: Vietoj tuščio `optim.Adam` optimizatoriaus, įveskite tikrus treniruojamus parametrus. Pavyzdžiui, galite optimizuoti pačias afininės transformacijos matricas.
3.  **Optimizuokite PSNR skaičiavimą**: Prieš skaičiuojant PSNR, normalizuokite įvesties ir išvesties duomenis iki to paties diapazono (pvz., [0, 1]).
4.  **Apsaugos mechanizmai**: Mutacijos funkcijose įdiekite griežtesnes patikras, kad būtų išvengta loginių klaidų ir netinkamų parametrų (pvz., negatyvių `block_size` reikšmių).

Įgyvendinus šiuos pakeitimus, jūsų sistema taps ne tik puikiai struktūrizuotu, bet ir realiai veikiančiu fraktalinės kompresijos įrankiu. Ar turėtumėte konkrečių klausimų apie fraktalinės kompresijos logikos įgyvendinimą CUDA branduoliuose?

```python
# --- Sistemos konstantos ---
class SystemConfig:
    DEVICE = "cuda" if torch.cuda.is_available() else "cpu"
    INIT_INPUT_SHAPE = (1, 64, 64)  # Standartinis įvesties dydis
    QAT_BITS = [1, 2, 4, 8, 16, 32]

class LUTType(Enum):

    DENSE = auto()
    HYBRID = auto()

@dataclass

    block_sizes: List[int] = field(default_factory=lambda: [4, 8, 16, 32])
    ifs_coeffs: np.ndarray = field(default_factory=lambda: np.random.uniform(-1, 1, 10))

# --- CUDA branduoliai ---
FRACTAL_CUDA = load_inline(
    name='fractal_ops',
    cpp_sources='',
    cuda_sources='''
    __device__ float affine_transform(float x, float scale, float offset) {
        return scale * x + offset;
    }
    
    __global__ void fractal_encode_kernel(
        const float* input, float* transforms, 
        int input_size, int block_size, int n_transforms
    ) {
        int idx = blockIdx.x * blockDim.x + threadIdx.x;
        if (idx >= n_transforms) return;
        
        // Supaprastinta transformacijų paieška
        transforms[idx*3] = 0.5f;  // scale
        transforms[idx*3+1] = 0.1f; // offset
        transforms[idx*3+2] = 0.0f; // rotation (supaprastinta)
    }
    
    __global__ void fractal_decode_kernel(
        float* output, const float* transforms,
        int output_size, int block_size, int max_iter
    ) {
        int idx = blockIdx.x * blockDim.x + threadIdx.x;
        if (idx >= output_size) return;
        
        float val = 0.0f;
        for (int i = 0; i < max_iter; i++) {
            int transform_idx = idx % (output_size / block_size);
            val = affine_transform(
                val,
                transforms[transform_idx*3],
                transforms[transform_idx*3+1]
            );
        }
        output[idx] = val;
    }
    ''',
    functions=['fractal_encode_kernel', 'fractal_decode_kernel'],
    verbose=False
)

# --- Fraktalinis enkoderis/dekoderis ---
class FractalProcessor:
    def __init__(self, params: FractalParams):
        self.params = params
        self.device = SystemConfig.get_device()
        self._setup_dirs()
        
    def _setup_dirs(self):
        SystemConfig.MODEL_SAVE_DIR.mkdir(exist_ok=True)
        SystemConfig.ACTUATOR_CACHE_DIR.mkdir(exist_ok=True)
    
    def _get_actuator_cache_path(self, data_hash: str) -> Path:
        return SystemConfig.ACTUATOR_CACHE_DIR / f"{data_hash}.pt"
    
    def _quantize(self, tensor: torch.Tensor, bits: int) -> torch.Tensor:
        """Adaptyvus kvantizavimo metodas"""
        if bits >= 32:
            return tensor
            
        min_val = tensor.min()
        max_val = tensor.max()
        scale = (max_val - min_val) / (2**bits - 1)
        
        quantized = ((tensor - min_val) / (scale + 1e-8)).round()
        return quantized * scale + min_val
    
    def _create_lut(self, data: torch.Tensor) -> torch.Tensor:
        """Sukuria LUT pagal įvesties duomenis"""
        n_blocks = math.prod(data.shape) // np.prod(self.params.block_sizes)
        transforms = torch.zeros(n_blocks * 3, device=self.device)
        
        threads = 256
        blocks = (n_blocks + threads - 1) // threads
        
        FRACTAL_CUDA.fractal_encode_kernel[blocks, threads](
            data.flatten(),
            transforms,
            math.prod(data.shape),
            np.prod(self.params.block_sizes),
            n_blocks
        )
        
        quantized = self._quantize(transforms, 8)  # Fiksuotas bitų skaičius LUT
        
        if self.params.lut_type == LUTType.SPARSE:
            return quantized.to_sparse()
        return quantized
    
    def encode(self, data: torch.Tensor) -> torch.Tensor:
        """Fraktalinis kodavimas su actuator cache'u"""
        data_hash = hashlib.sha256(data.cpu().numpy().tobytes()).hexdigest()
        cache_path = self._get_actuator_cache_path(data_hash)
        
        if cache_path.exists():
            actuator = torch.load(cache_path)
            logger.info(f"Using cached actuator for data hash: {data_hash[:8]}")
            return actuator
            
        actuator = self._create_lut(data)
        torch.save(actuator, cache_path)
        return actuator
    
    def decode(self, actuator: torch.Tensor, output_shape: Tuple[int,...]) -> torch.Tensor:
        """Fraktalinis dekodavimas"""
        output = torch.zeros(output_shape, device=self.device)
        
        if actuator.is_sparse:
            actuator = actuator.to_dense()
            
        threads = 256
        output_size = math.prod(output_shape)
        blocks = (output_size + threads - 1) // threads
        
        FRACTAL_CUDA.fractal_decode_kernel[blocks, threads](
            output.flatten(),
            actuator,
            output_size,
            np.prod(self.params.block_sizes),
            self.params.max_iterations
        )
        
        return output.reshape(output_shape)

# --- Genetinis algoritmas ---
class GeneticOptimizer:
    def __init__(self, population_size: int = 20):
        self.population = []
        self.population_size = population_size
        self.current_generation = 0
        self.best_fitness = -float('inf')
        self._init_population()
        
    def _init_population(self):
        for _ in range(self.population_size):
            genome = {
                'fractal': FractalParams(),
                'training': TrainingParams(),
                'fitness': -1.0,
                'vram_usage': 0.0,
                'psnr': 0.0,
                'id': str(random.getrandbits(128))
            }
            self.population.append(genome)
    
    async def evaluate_population(self, data_loader):
        """Asinchroninis populiacijos vertinimas"""
        tasks = []
        for genome in self.population:
            task = self._evaluate_genome(genome, data_loader)
            tasks.append(task)
            
        await asyncio.gather(*tasks)
        self.current_generation += 1
        
    async def _evaluate_genome(self, genome, data_loader):
        """Vertina genomą naudojant fraktalinį autoenkoderį"""
        try:
            processor = FractalProcessor(genome['fractal'])
            optimizer = optim.Adam([], lr=genome['training'].learning_rate)
            
            total_psnr = 0.0
            batch_count = 0
            
            for batch in data_loader:
                batch = batch.to(SystemConfig.get_device())
                
                # Kodavimas/dekodavimas
                encoded = processor.encode(batch)
                decoded = processor.decode(encoded, batch.shape)
                
                # PSNR skaičiavimas
                batch_psnr = peak_signal_noise_ratio(
                    batch.cpu().numpy(),
                    decoded.cpu().numpy(),
                    data_range=1.0
                )
                total_psnr += batch_psnr
                batch_count += 1
                
                # VRAM naudojimas
                genome['vram_usage'] = torch.cuda.memory_allocated() / (1024 ** 2)
                
                if genome['vram_usage'] > SystemConfig.VRAM_LIMIT_MB * 0.8:
                    torch.cuda.empty_cache()
                    genome['fitness'] *= 0.9  # Penalty už VRAM viršijimą
                    break
            
            genome['psnr'] = total_psnr / batch_count
            genome['fitness'] = self._calculate_fitness(genome)
            
        except Exception as e:
            logger.error(f"Genomo vertinimo klaida: {str(e)}")
            genome['fitness'] = -1.0
    
    def _calculate_fitness(self, genome) -> float:
        """Skaičiuoja fitnesą pagal PSNR, VRAM naudojimą ir kitus parametrus"""
        psnr_score = genome['psnr'] / 50.0  # Normalizuotas iki [0,1]
        vram_score = 1.0 - (genome['vram_usage'] / SystemConfig.VRAM_LIMIT_MB)
        complexity_score = 1.0 - (len(genome['fractal'].block_sizes) / 10.0)
        
        weights = {
            'psnr': 0.6,
            'vram': 0.3,
            'complexity': 0.1
        }
        
        return (
            psnr_score * weights['psnr'] +
            vram_score * weights['vram'] +
            complexity_score * weights['complexity']
        )
    
    def evolve(self):
        """Atlieka genetinę evoliuciją"""
        # Rūšiuojame pagal fitnesą
        self.population.sort(key=lambda x: x['fitness'], reverse=True)
        
        # Išsaugome geriausią genomą
        if self.population[0]['fitness'] > self.best_fitness:
            self.best_fitness = self.population[0]['fitness']
            self._save_best_genome()
        
        # Parenkame tėvus (elitizmas)
        parents = self.population[:int(self.population_size * 0.3)]
        
        # Sukuriame naują populiaciją
        new_population = parents.copy()
        
        # Kryžminimas ir mutacija
        while len(new_population) < self.population_size:
            parent1, parent2 = random.choices(parents, k=2)
            child = self._crossover(parent1, parent2)
            child = self._mutate(child)
            new_population.append(child)
        
        self.population = new_population
    
    def _crossover(self, parent1, parent2):
        """Sukuria vaiką iš dviejų tėvų"""
        child = {
            'fractal': FractalParams(),
            'training': TrainingParams(),
            'fitness': -1.0,
            'vram_usage': 0.0,
            'psnr': 0.0,
            'id': str(random.getrandbits(128))
        }
        
        # Kryžminame fraktalinius parametrus
        child['fractal'].block_sizes = random.choice([
            parent1['fractal'].block_sizes,
            parent2['fractal'].block_sizes
        ])
        
        # Kryžminame treniravimo parametrus
        child['training'].learning_rate = np.mean([
            parent1['training'].learning_rate,
            parent2['training'].learning_rate
        ])
        
        return child
    
    def _mutate(self, genome):
        """Atlieka mutacijas genome"""
        # Mutuojame blokų dydžius
        if random.random() < 0.1:
            idx = random.randint(0, len(genome['fractal'].block_sizes)-1)
            genome['fractal'].block_sizes[idx] += random.choice([-2, -1, 1, 2])
            genome['fractal'].block_sizes[idx] = max(2, genome['fractal'].block_sizes[idx])
        
        # Mutuojame mokymosi greitį
        if random.random() < 0.1:
            genome['training'].learning_rate *= random.uniform(0.5, 1.5)
            genome['training'].learning_rate = np.clip(
                genome['training'].learning_rate, 1e-5, 1e-2
            )
        
        return genome
    
    def _save_best_genome(self):
        """Išsaugo geriausią genomą"""
        best = max(self.population, key=lambda x: x['fitness'])
        save_path = SystemConfig.MODEL_SAVE_DIR / f"gen_{self.current_generation}.json"
        
        with open(save_path, 'w') as f:
            json.dump({
                'fractal_params': vars(best['fractal']),
                'training_params': vars(best['training']),
                'fitness': best['fitness'],
                'psnr': best['psnr'],
                'vram_usage': best['vram_usage']
            }, f, indent=2)

# --- Pramoninė sistema ---
class FractalAutoencoderSystem:
    def __init__(self):
        self.optimizer = GeneticOptimizer()
        self.data_loader = self._create_dummy_loader()
        
    def _create_dummy_loader(self):
        """Sukuria dummy duomenų rinkinį testavimui"""
        dummy_data = torch.randn(100, *SystemConfig.INIT_INPUT_SHAPE)
        dataset = torch.utils.data.TensorDataset(dummy_data)
        return torch.utils.data.DataLoader(
            dataset, batch_size=32, shuffle=True
        )
    
    async def run_optimization(self, generations: int = 50):
        """Vykdo genetinę optimizaciją"""
        logger.info("Pradedama genetinė optimizacija")
        
        for gen in range(generations):
            start_time = time.time()
            
            await self.optimizer.evaluate_population(self.data_loader)
            self.optimizer.evolve()
            
            best = max(self.optimizer.population, key=lambda x: x['fitness'])
            logger.info(
                f"Karta {gen+1}/{generations} | "
                f"Geriausias fitnesas: {best['fitness']:.4f} | "
                f"PSNR: {best['psnr']:.2f} dB | "
                f"VRAM: {best['vram_usage']:.2f} MB"
            )
            
            # Periodinis valymas
            if gen % 5 == 0:
                torch.cuda.empty_cache()
                
            # Adaptyvus generacijų intervalas
            elapsed = time.time() - start_time
            if elapsed < 1.0:
                await asyncio.sleep(1.0 - elapsed)

# --- Pagrindinė funkcija ---
async def main():
    try:
        system = FractalAutoencoderSystem()
        await system.run_optimization(generations=100)
        
    except Exception as e:
        logger.critical(f"Kritinė klaida: {str(e)}")
        traceback.print_exc()

if __name__ == "__main__":
    asyncio.run(main())
```
