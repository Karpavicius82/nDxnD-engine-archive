### **1. Sistemos apžvalga**

Jūsų kodas aprašo sudėtingą genetinį algoritmą, skirtą evoliucionuoti **hibridinius neuroninius tinklus (HNN)**. Šie tinklai sujungia tradicinius giliojo mokymosi modelius su fraktaline bei peržvalgos lentelių (**Look-Up Table, LUT**) technologija. Pagrindiniai sistemos komponentai:

  * **`Genome`**: Tai evoliucionuojantis objektas, apibrėžiantis HNN architektūrą (sluoksnius, mokymosi parametrus), taip pat naujus, eksperimentinius parametrus, tokius kaip **Iteruojamų Funkcijų Sistemų (IFS)** koeficientai, fraktalinių blokų dydžiai ir LUT konfigūracija.
  * **`HNN_Core`**: Pagrindinis tinklo modelis, kuris vietoj įprastų konvoliucinių sluoksnių naudoja **fraktalinį kodavimą ir dekodavimą**. Tai leidžia įterpti fraktalinę geometriją ir LUT į kompresijos procesą.
  * **`GeneticArchitect`**: Genetinis algoritmas, kuris apima populiacijos inicializavimą, genomo vertinimą, tėvų atranką, kryžminimą ir mutaciją.
  * **Kompresijos mechanizmai**: Kompresija atliekama per du pagrindinius etapus:
    1.  **Fraktalinis kodavimas**: Naudojant hierarchinį blokų dalijimą ir afininę transformaciją su LUT.
    2.  **Kiekinis apdorojimas (Quantization)**: Svarbi `HNN_Core` dalis, kuri sumažina skaitinių reikšmių tikslumą, taip tiesiogiai mažindama modelio ir latentinės erdvės dydį.

### **2. Kompresijos efektyvumo vertinimas**

Jūsų sukurtas kodas siekia kompresijos keliais lygmenimis, o tai yra teisingas, modernus metodas, naudojamas pramonėje. Tačiau vertinant **`HNN_Core`** kompresijos efektyvumą, randama ir stipriųjų, ir silpnųjų pusių.

#### **Stipriosios pusės**

  * **Įvairiapusė kompresija**: Derinami keli kompresijos metodai:
      * **Fraktalinė kompresija**: Vaizdo (ar kitų duomenų) informacijai užkoduoti pasitelkiama fraktalų geometrija, kuri leidžia efektyviai atkurti sudėtingas struktūras iš mažesnio kiekio parametrų.
      * **Kvantavimas (Quantization)**: Sumažinamas duomenų ir modelio svorių tikslumas, o tai yra vienas iš pagrindinių **Mašininio mokymosi (ML)** kompresijos metodų.
      * **LUT naudojimas**: Peržvalgos lentelės leidžia greičiau rasti atitinkamą transformaciją, eliminuojant skaičiavimus.
  * **Automatinė optimizacija**: Genetinis algoritmas automatiškai parenka geriausius kompresijos parametrus (fraktalinius blokus, LUT dydį, kvantavimo lygius), o tai yra geriau nei rankinė optimizacija.
  * **`VRAM` valdymas**: Įdiegta patikrinimo ir valymo funkcija, kas yra itin svarbu dirbant su dideliais modeliais ir ribota VRAM atmintimi.

#### **Silpnosios pusės ir tobulinimo sritys**

  * **Tikslumo (PSNR) ir kompresijos santykis**: Kode yra apskaičiuojamas PSNR, o tai yra geras metrikas, bet pramonėje dažnai naudojama kreivė **RD-kreivė (Rate-Distortion curve)**, kuri parodo kompresijos santykį (R) ir iškraipymą (D) (PSNR yra iškraipymo metrikas). Be šios kreivės sunku objektyviai įvertinti, ar kompresija yra geresnė nei, pavyzdžiui, **JPEG** ar **H.264**.
  * **Kodo optimizavimas**: Nors **Numba CUDA** naudojimas yra puikus žingsnis, fraktalinis dekodavimo branduolys (kernel) atrodo per daug supaprastintas. `ifs_iterate_kernel` funkcijoje naudojami `blocks` ir `coeffs` kintamieji, bet rezultatai įrašomi į `out.numpy()`, kas nėra tinkama, nes `Numba` branduolys dirba tik su `numpy` masyvais, o grįžtamos reikšmės vėl konvertuojamos į `torch` tensorius. Tai sukuria papildomą režijos (overhead) naštą ir gali būti neefektyvu.
  * **Fraktalinio ir LUT ryšys**: Ryšys tarp fraktalinio kodo ir LUT atrodo daugiau teorinis nei praktiškas. `fractal_encode` naudoja `idx` iš modulio operacijos, kas gali sukelti **kolizijas** ir sumažinti kompresijos efektyvumą. Reikėtų detalesnio, pramoninio lygio fraktalinio kodo (pvz., **Fractal Image Compression**) įgyvendinimo.
  * **Tradicinių ir HNN metodų palyginimas**: Trūksta ataskaitos, kuri parodytų, kaip jūsų HNN kompresija atlieka lyginant su tradiciniais metodais. Tam reikia sukurti etaloninius modelius (baseline) ir atlikti palyginamuosius testus.

class SystemConfig:
    device: str = "cuda" if torch.cuda.is_available() else "cpu"
    initial_input_shape: Tuple[int, ...] = (1, 8, 8)  # Pradinis 2D, bet nD

# --- Genomo klasė su naujais fraktaliniais/LUT params ---
class LayerDefinition:

class Genome:

    def _normalize_fitness_weights(self):

        if total_weight > 0:
            for key in self.fitness_weights:
                self.fitness_weights[key] /= total_weight

# --- HNN Modelis su fraktaliniu pakeitimu ir LUT ---
class HNN_Core(nn.Module):
    def __init__(self, encoder: nn.Module, decoder: nn.Module, latent_dim: int, genome: Genome, pre_flatten_shapes: List[Tuple[int, ...]]):
        super().__init__()
        self.encoder = encoder
        self.decoder = decoder
        self.latent_dim = latent_dim
        self.genome = genome
        self.pre_flatten_shapes = pre_flatten_shapes
        self.device = torch.device(SystemConfig.device)
        self.supported_qat_bits = SystemConfig.supported_qat_bits
        self.initial_input_shape = SystemConfig.initial_input_shape
        self.lut = self._init_lut()  # LUT inicializacija

    def _init_lut(self) -> torch.Tensor:
        # Sparse/dense LUT inicializacija precompute
        if self.genome.lut_type == 'sparse':
            indices = torch.randint(0, self.genome.lut_size, (2, self.genome.lut_size // 2), device=self.device)
            values = torch.rand(self.genome.lut_size // 2, device=self.device)
            return torch.sparse.FloatTensor(indices, values, size=(self.genome.lut_size, self.genome.lut_size)).to(self.device)
        else:
            return torch.rand(self.genome.lut_size, self.genome.lut_size, device=self.device)

    def custom_quantize(self, tensor: torch.Tensor, bits: int) -> torch.Tensor:
        if bits >= 32:
            return tensor
        max_val = tensor.abs().max()
        if max_val == 0:
            return tensor
        scale = (2.0**bits - 1.0) / (2.0 * max_val)
        zero_point = 2.0**(bits-1)
        quantized = torch.round(tensor * scale + zero_point)
        return (quantized - zero_point) / scale

    @cuda.jit
    def ifs_iterate_kernel(blocks, coeffs, out, max_iter):
        # Numba CUDA kernel IFS iteracijoms (parallel per bloką)
        tx = cuda.threadIdx.x
        ty = cuda.blockIdx.x
        if tx < blocks.shape[0] and ty < blocks.shape[1]:
            z_real = blocks[tx, ty, 0]
            z_imag = blocks[tx, ty, 1]
            for i in range(max_iter):
                z_real_new = z_real**2 - z_imag**2 + coeffs[0]
                z_imag_new = 2 * z_real * z_imag + coeffs[1]
                if z_real_new**2 + z_imag_new**2 > 4:
                    break
                z_real, z_imag = z_real_new, z_imag_new
            out[tx, ty] = i

    def fractal_encode(self, x: torch.Tensor) -> torch.Tensor:
        # Fraktalinis enkoderis su HHN hierarchija (blokų dalinimas)
        N = SystemConfig.get_N_dims()
        blocks = self._hierarchical_block_split(x, self.genome.fractal_blocks)  # nD blokai
        latent = []
        for block in blocks:
            # LUT lookup + IFS transformacija
            idx = int(np.prod(block.shape) % self.genome.lut_size)
            transform = self.lut[idx] if self.genome.lut_type == 'dense' else self.lut._values()[idx % len(self.lut._values())]
            latent.append(transform @ block.view(-1))  # Afinė transformacija
        latent = torch.stack(latent).mean(dim=0)
        self.genome.fractal_actuator = latent  # Išsaugoti actuator
        torch.save(self.genome.fractal_actuator, f'actuator_{self.genome.id}.pth')
        return self.custom_quantize(latent, random.choice(self.genome.layer_bits['encoder']))

    def fractal_decode(self, latent: torch.Tensor) -> torch.Tensor:
        if self.genome.fractal_actuator is None:
            self.genome.fractal_actuator = torch.load(f'actuator_{self.genome.id}.pth')
        # Dekodavimas per išsaugotą actuator (IFS iteracijos CUDA)
        out_shape = (latent.shape[0], *self.initial_input_shape[1:])
        out = torch.zeros(out_shape, device=self.device)
        blocks = np.random.rand(latent.shape[0], latent.shape[1], 2)  # Dummy blocks nD
        coeffs = self.genome.ifs_coeffs[:2]  # Pavyzdiniai coeffs
        self.ifs_iterate_kernel[latent.shape[0], latent.shape[1]](blocks, coeffs, out.numpy(), 20)  # CUDA call
        return torch.from_numpy(out).to(self.device)

    def _hierarchical_block_split(self, x: torch.Tensor, blocks: List[int]) -> List[torch.Tensor]:
        # HHN hierarchinis dalinimas nD (quadtree/octtree generalizuotas)
        split_blocks = []
        current = x
        for size in blocks:
            # nD split (hiperkubai)
            for i in range(0, current.shape[1], size):
                for j in range(0, current.shape[2], size):
                    block = current[:, i:i+size, j:j+size]
                    split_blocks.append(block)
            current = torch.stack(split_blocks).mean(dim=0)  # Hierarchinis reduce
        return split_blocks

    def forward(self, x: torch.Tensor) -> Tuple[torch.Tensor, torch.Tensor]:
        if len(x.shape) == len(self.initial_input_shape):
            x = x.unsqueeze(0)
        encoded = self.fractal_encode(x)
        decoded = self.fractal_decode(encoded)
        return encoded, decoded

        return activations.get(name, nn.Identity)()

    @staticmethod

        try:
            N = SystemConfig.get_N_dims()
            # Fraktalinis hibridas – vietoj conv, naudoti fractal_encode/decode
            encoder_modules = []  # Placeholder, fraktalinis per forward

# --- Genetinis Algoritmas su LUT/PSNR ---
class GeneticArchitect:
    def __init__(self, config: SystemConfig, initial_input_shape: Tuple[int, ...]):
        self.config = config
        self.population: List[Genome] = []
        self.initial_input_shape = initial_input_shape
        self.last_vram_clear_time = time.time()
        self.mutation_rate = 0.1
        self.crossover_rate = 0.7
        self.best_fitness_history = []
        self.history = []
        self.surrogate = None

    async def initialize_population(self, population_size: int):
        self.population = []
        avg_clusters = np.mean([g.n_clusters for g in self.population] or [5])
        random_params = [np.random.rand(10) for _ in range(100)]
        kmeans = KMeans(n_clusters=avg_clusters)
        kmeans.fit(random_params)
        labels = kmeans.labels_
        for cluster in range(avg_clusters):
            cluster_params = [p for p, l in zip(random_params, labels) if l == cluster]
            for _ in range(population_size // avg_clusters):
                genome = Genome()
                genome.params = random.choice(cluster_params) if cluster_params else np.random.rand(10)
                self.population.append(genome)
        logger.info(f"Populiacija inicializuota su {len(self.population)} genomų.")

    def _check_vram_limit(self, genome: Genome):
        if torch.cuda.is_available():
            torch.cuda.empty_cache()
            genome.vram_used_mb = torch.cuda.memory_allocated(0) / (1024 ** 2)
            if genome.vram_used_mb > genome.vram_limit_mb * genome.vram_warning_threshold:
                logger.warning("VRAM viršytas, valoma.")

    def _compute_psnr(self, original: torch.Tensor, decoded: torch.Tensor) -> float:
        mse = nn.MSELoss()(original, decoded)
        if mse == 0:
            return float('inf')
        max_val = original.max()
        return 20 * math.log10(max_val / math.sqrt(mse))

    async def evaluate_genome(self, genome: Genome) -> None:
        device = torch.device(self.config.device)
        # Dummy data: Fractal color image (Mandelbrot scipy)
        dummy_data = self._generate_fractal_data(genome.batch_size)
        dummy_data = dummy_data.to(device)

        hnn_model = HNNBuilder.build_model_from_genome(genome, self.initial_input_shape, device)
        if hnn_model is None:
            genome.fitness = -float('inf')
            return

        criterion = nn.MSELoss()
        optimizer = optim.Adam(hnn_model.parameters(), lr=genome.learning_rate)

        hnn_model.train()
        for _ in range(genome.training_epochs):
            optimizer.zero_grad()
            encoded, decoded = hnn_model(dummy_data)
            loss = criterion(decoded, dummy_data)
            loss.backward()
            optimizer.step()
            self._check_vram_limit(genome)

        hnn_model.eval()
        with torch.no_grad():
            encoded, decoded = hnn_model(dummy_data)
            val_loss = criterion(decoded, dummy_data)
            genome.loss = val_loss.item()
            genome.psnr_score = self._compute_psnr(dummy_data, decoded)  # PSNR

        if genome.psnr_score < genome.psnr_threshold:
            # Auto-mutate actuator/LUT
            genome.fractal_actuator = None  # Reset
            genome.lut = hnn_model._init_lut()  # Re-precompute

        # Compression ratio + VRAM score
        input_size_bits = math.prod(self.initial_input_shape) * 32
        latent_size_bits = genome.latent_dim_flat * sum(genome.layer_bits['encoder']) / len(genome.layer_bits['encoder'])
        genome.compression_ratio = input_size_bits / latent_size_bits if latent_size_bits > 0 else 0.0
        vram_score = 1.0 - genome.vram_used_mb / genome.vram_limit_mb

        # Fitness su PSNR
        normalized_weights = {k: v / sum(genome.fitness_weights.values()) for k, v in genome.fitness_weights.items()}
        genome.fitness = (genome.accuracy * normalized_weights["hnn_accuracy"] +
                          vram_score * normalized_weights["vram_efficiency"] +
                          genome.compression_ratio * normalized_weights["compression_ratio_impact"] +
                          genome.psnr_score / 100 * normalized_weights["psnr_score"])

        self.history.append((np.array([genome.latent_dim_flat, genome.psnr_score]), genome.fitness))
        del hnn_model
        torch.cuda.empty_cache()

    def _generate_fractal_data(self, batch_size: int) -> torch.Tensor:
        # Generuoti Mandelbrot fractal data (scipy/matplotlib colormap)
        x = np.linspace(-2, 1, self.initial_input_shape[1])
        y = np.linspace(-1.5, 1.5, self.initial_input_shape[2])
        c_real, c_imag = np.meshgrid(x, y)
        z_real = np.zeros_like(c_real)
        z_imag = np.zeros_like(c_imag)
        mask = np.ones_like(c_real, dtype=bool)
        for _ in range(50):
            z_real[mask] = z_real[mask]**2 - z_imag[mask]**2 + c_real[mask]
            z_imag[mask] = 2 * z_real[mask] * z_imag[mask] + c_imag[mask]
            mask = (z_real**2 + z_imag**2 < 4)
        img = np.log(np.maximum(z_real**2 + z_imag**2, 1e-10))
        colormap = mcolors.LinearSegmentedColormap('fractal', np.array([[0,0,0],[1,0,0],[0,1,0],[0,0,1]]))
        img = colormap(img / img.max())
        return torch.from_numpy(img[..., :1]).permute(2, 0, 1).repeat(batch_size, 1, 1, 1).float()  # Batch x 1 x H x W

    # Likusios funkcijos (train_surrogate, apply_niche_penalty, select_parents, crossover, mutate) lieka panašios, bet su naujais params mutate (pvz., lut_size *= random.uniform(0.9, 1.1))

    def mutate(self, genome: Genome) -> None:
        # ... (ankstesnis mutate)
        # Nauji: Fraktaliniai/LUT mutate
        genome.fractal_blocks = [max(2, min(32, b + random.choice([-2, 2]))) for b in genome.fractal_blocks]
        genome.ifs_coeffs += random.uniform(-genome.mutation_strength, genome.mutation_strength)
        genome.lut_size = max(1024, min(int(1e6), int(genome.lut_size * random.uniform(0.9, 1.1))))
        genome.psnr_threshold += random.uniform(-2, 2)
        genome.actuator_reuse_limit += random.choice([-1, 1])
        if random.random() < 0.1:
            genome.lut_type = random.choice(['sparse', 'dense'])

async def main():
    system_config = SystemConfig()
