-----

Absoliuti teisybė\! **LUT (Look-Up Table)** pridėjimas prie dabartinės architektūros yra puikus žingsnis, kad pasiektume jūsų viziją apie tiesioginę hologramos sąveiką su AI, išlaikant esminę HNN nD prigimtį. Iš tiesų, aš jau buvau įtraukęs LUT į ankstesnes versijas `HNN_QuantLUT_Core` modulyje, tačiau siekiant sutelkti dėmesį į nD tenzoriaus pritaikymą, jis buvo supaprastintas.

Dabar, kai grįžtame prie idėjos integruoti **LUT tiesiogiai į nD hologramos generavimą**, tai leidžia:

1.  **Išlaikyti "kvantuotos hologramos" esmę:** LUT naudojimas reiškia, kad įvesties duomenys yra kvantuojami į diskretinius indeksus, kurie atspindi unikalias holografines "daleles". Kiekvienas toks indeksas susiejamas su išmoktu įterpimu (embedding), kuris vėliau sudaro hologramos tūrį. Tai yra tikrasis "holografinis" principas, kai informacija yra paskirstyta ir atkuriami išmokti modeliai.
2.  **Suteikti AI gilesnį supratimą apie hologramos "prigimtį":** AI modelis nebeturi mokytis kvantavimo logikos iš nulio. LUT jau pateikia išmoktus, kvantuotus reprezentacijas, su kuriomis AI gali dirbti efektyviau. Tai tarsi AI gauna "paruoštą" holografinę informaciją.
3.  **Optimizuoti atmintį:** LUT gali būti efektyvus būdas saugoti ir pasiekti didelius, bet diskrečius informacijos kiekius, kas yra svarbu, kai kalbame apie dideles nD matricas ir ribotą VRAM.

### 2\. `hnn_nd_core.py` (Atnaujintas su LUT)

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from config import HNN_LUT_CONFIG, SYSTEM_CONFIG

class HNN_nD_Core(nn.Module):
    def __init__(self, n_dims: int, dim_size: int, quant_bits: int, 
                 lut_output_embedding_dim: int):
        super().__init__()
        self.n_dims = n_dims
        self.dim_size = dim_size
        self.quant_range = 2**quant_bits 
        self.lut_output_embedding_dim = lut_output_embedding_dim

        # --- Pagrindinis LUT (Look-Up Table) ---
        # Kiekvienam iš 2^quant_bits galimų indeksų bus sukurtas embedding
        self.holographic_lut = nn.Embedding(
            num_embeddings=self.quant_range, 
            embedding_dim=self.lut_output_embedding_dim
        )

        # --- Įvesties Apdorojimas ir Hologramos Generavimas ---
        # Šis sluoksnis ims sujungtus LUT embeddings ir paverčia juos į nD tenzorių.
        # Įvesties dydis: timesteps * input_features * lut_output_embedding_dim
        # Išvesties dydis: dim_size ** n_dims
        
        input_for_hologram_gen = (
            HNN_LUT_CONFIG["timesteps"] * HNN_LUT_CONFIG["input_features"] * self.lut_output_embedding_dim
        )
        
        self.hologram_generator_linear = nn.Linear(
            input_for_hologram_gen, 
            self.dim_size ** self.n_dims 
        )

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        # 1. Finansinių duomenų normalizavimas ir kvantavimas į 8 bitų indeksus
        # Tai atliekama taip pat, kaip ir anksčiau, kad duomenys taptų tinkami LUT.
        min_val = x.min(dim=-1, keepdim=True)[0].min(dim=-2, keepdim=True)[0]
        max_val = x.max(dim=-1, keepdim=True)[0].max(dim=-2, keepdim=True)[0]
        normalized_data = (x - min_val) / (max_val - min_val + 1e-6) 
        quantized_indices = (normalized_data * (self.quant_range - 1)).round().long() 
        
        # 2. Naudojame LUT, kad gautume "holografinius" įterpimus
        # holographic_embeddings_from_lut forma: 
        # (batch_size, timesteps, input_features, lut_output_embedding_dim)
        holographic_embeddings_from_lut = self.holographic_lut(quantized_indices)
        
        # Sujungiame visus embeddings į vieną ilgą vektorių vienam pavyzdžiui
        # flat_holographic_embeddings forma: 
        # (batch_size, timesteps * input_features * lut_output_embedding_dim)
        flat_holographic_embeddings = holographic_embeddings_from_lut.view(
            holographic_embeddings_from_lut.size(0), -1
        )
        
        # 3. Hologramos "generavimas" ir pertvarkymas į tikrą nD tenzorių
        # hologram_flat forma: (batch_size, dim_size ** n_dims)
        hologram_flat = self.hologram_generator_linear(flat_holographic_embeddings)
        
        # Pertvarkome išlygintą tenzorių į tikrą nD tenzorių
        # nD_hologram forma: (batch_size, dim_size, dim_size, ..., dim_size)
        hologram_shape = (hologram_flat.size(0),) + (self.dim_size,) * self.n_dims
        nD_hologram = hologram_flat.view(hologram_shape)
        
        return nD_hologram
```

### 3\. `nd_ai_head.py` (Be pakeitimų)

Ši klasė lieka nepakeista, nes ji jau buvo sukurta taip, kad tiesiogiai priimtų ir apdorotų nD tenzorių iš `HNN_nD_Core`.

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
from config import ND_AI_HEAD_CONFIG, HNN_LUT_CONFIG, SYSTEM_CONFIG

class nD_AI_Head(nn.Module):
    def __init__(self, n_dims: int, dim_size: int):
        super().__init__()
        self.n_dims = n_dims
        self.dim_size = dim_size
        self.output_actions = ND_AI_HEAD_CONFIG["output_actions"]
        self.hidden_project_dim = ND_AI_HEAD_CONFIG["hidden_project_dim"]

        # Šiame modulyje holografinio tenzoriaus agregavimui naudojamos paprastos operacijos
        # be tiesioginių konvoliucijų, siekiant išlaikyti nD sąveikos principą.
        # `final_embedding_dim` yra nustatytas pagal tai, kaip buvo sugeneruoti LUT embeddings,
        # nes galiausiai viskas susiveda į šį matmenį per `HNN_nD_Core` ir projekciją.
        # Tikrasis dydis po vidurkio skaičiavimo yra n_dims * dim_size.
        projection_input_dim = self.n_dims * self.dim_size
        
        self.dynamic_projection_layer = nn.Linear(projection_input_dim, self.hidden_project_dim).to(SYSTEM_CONFIG["device"])
        
        self.decision_linear = nn.Sequential(
            nn.Linear(self.hidden_project_dim, self.hidden_project_dim // 2),
            nn.ReLU(),
            nn.Linear(self.hidden_project_dim // 2, self.output_actions)
        )

    def forward(self, nD_hologram: torch.Tensor) -> tuple[str, float, float, torch.Tensor]:
        # nD_hologram forma: (batch_size, dim_size, dim_size, ..., dim_size)
        
        # Atliekame svertinį vidurkį per kiekvieną dimensiją, išskyrus batch.
        # Tai išlaiko nD formą kuo ilgiau ir leidžia "projiektuoti" informaciją.
        averaged_dims = []
        # Iteruojame per kiekvieną N dimensiją (po batch_size)
        for i in range(self.n_dims):
            # Sukuriame sąrašą dimensijų, per kurias norime imti vidurkį, išskyrus i-tąją.
            # Pvz., jei n_dims=3 ir i=0, tai imsime vidurkį per dimensijas 2 ir 3.
            dims_to_average = [d for d in range(1, self.n_dims + 1) if d != (i + 1)]
            
            # Paimame vidurkį ir tada pašaliname tas dimensijas.
            # Keepdim=True, kad išlaikytų tą dimensiją, tada squeeze, kad ją pašalintų.
            avg_dim_slice = torch.mean(nD_hologram, dim=dims_to_average, keepdim=False)
            averaged_dims.append(avg_dim_slice) 
        
        # Sujungiame gautas vidurkio projekcijas į vieną vektorių
        # Tai bus (batch_size, n_dims * dim_size)
        projected_hologram_vector = torch.cat(averaged_dims, dim=-1)

        # Projekcija į sprendimo erdvę
        final_holographic_embedding = self.dynamic_projection_layer(projected_hologram_vector)

        # Galutinis sprendimas
        logits = self.decision_linear(final_holographic_embedding)
        probabilities = F.softmax(logits, dim=-1) 

        action_idx = torch.argmax(probabilities, dim=-1).item()
        action_map = {0: "BUY", 1: "SELL", 2: "HOLD"}
        action = action_map[action_idx]
        confidence = probabilities[0, action_idx].item()
        
        grok_confidence = confidence 
        
        return action, confidence, grok_confidence, logits
```

### 4\. `unified_hnn_ai_fabric.py` (Be pakeitimų)

Ši klasė lieka nepakeista, nes jos užduotis yra tiesiog sujungti HNN branduolį ir AI galvą.

```python
import torch.nn as nn
from hnn_nd_core import HNN_nD_Core # Naujas HNN branduolys
from nd_ai_head import nD_AI_Head # Nauja AI galva
from config import HNN_LUT_CONFIG, SYSTEM_CONFIG

class UnifiedHNN_AIFabric(nn.Module):
    def __init__(self, fabric_params: dict):
        super().__init__()
        
        # HNN nD branduolys
        self.hnn_core = HNN_nD_Core(
            n_dims=fabric_params["n_dims"],
            dim_size=fabric_params["dim_size"],
            quant_bits=HNN_LUT_CONFIG["quant_bits"],
            lut_output_embedding_dim=HNN_LUT_CONFIG["lut_output_embedding_dim"]
        )
        
        # nD AI galva, tiesiogiai priimanti nD hologramą
        self.ai_head = nD_AI_Head(
            n_dims=fabric_params["n_dims"], 
            dim_size=fabric_params["dim_size"] 
        )
        
    def forward(self, x: torch.Tensor) -> tuple[str, float, float, torch.Tensor]:
        # HNN nD branduolys sugeneruoja nD hologramą
        nD_hologram = self.hnn_core(x)

        # nD AI galva tiesiogiai apdoroja nD hologramą
        action, confidence, grok_confidence, logits = self.ai_head(nD_hologram)
        
        return action, confidence, grok_confidence, logits
```

### 5\. `main.py` (Be pakeitimų)

Šis failas veikia kaip pagrindinė vykdymo logika ir gali likti nepakeistas, nes `UnifiedHNN_AIFabric` sąsaja liko ta pati.

```python
import torch
import torch.nn as nn
import torch.optim as optim
import logging
import copy 
from tqdm import tqdm 
import numpy as np
import gc
import sys # For getting current module size
import random

from config import SYSTEM_CONFIG, GA_CONFIG, TRAINING_CONFIG, HNN_LUT_CONFIG, ND_AI_HEAD_CONFIG
from unified_hnn_ai_fabric import UnifiedHNN_AIFabric
from genetic_orchestrator import GeneticOrchestrator
from environment_simulator import EnvironmentSimulator

# Nustatome logavimą
logging.basicConfig(level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s', stream=sys.stdout)
logger = logging.getLogger(__name__)

def get_gpu_memory_usage():
    """Grąžina dabartinį GPU atminties naudojimą GB."""
    if torch.cuda.is_available():
        allocated = torch.cuda.memory_allocated() / (1024**3)
        reserved = torch.cuda.memory_reserved() / (1024**3)
        return allocated, reserved
    return 0.0, 0.0

def clean_gpu_memory():
    """Išvalo GPU atminties talpyklą."""
    if torch.cuda.is_available():
        torch.cuda.empty_cache()
    gc.collect() 

def main():
    device = SYSTEM_CONFIG["device"]
    logger.info(f"Sistema veikia su įrenginiu: {device}")
    if device == "cuda":
        total_gpu_mem = torch.cuda.get_device_properties(0).total_memory / (1024**3)
        logger.info(f"Bendras GPU VRAM: {total_gpu_mem:.2f} GB. Nustatytas VRAM limitas: {GA_CONFIG['vram_limit_gb']:.2f} GB.")
    
    ga_orchestrator = GeneticOrchestrator()
    env_simulator = EnvironmentSimulator()

    total_system_roi_history = []
    
    for generation in range(GA_CONFIG["num_generations"]):
        logger.info(f"\n--- Pradedama {generation + 1} evoliucijos karta iš {GA_CONFIG['num_generations']} ---")
        
        current_generation_feedback = [] 

        for individual_idx, individual in enumerate(tqdm(ga_orchestrator.population, desc=f"Karta {generation+1} Individualai")):
            clean_gpu_memory() 

            fabric_params = individual["params"]
            current_vram_usage_gb = 0.0

            try:
                current_system = UnifiedHNN_AIFabric(fabric_params).to(device)
                
                optimizer = optim.Adam(current_system.parameters(), lr=TRAINING_CONFIG["learning_rate"])
                criterion = nn.CrossEntropyLoss() 

                current_individual_roi = []
                current_individual_ai_conf = []
                current_individual_grok_conf = []

                for epoch in range(TRAINING_CONFIG["epochs_per_individual"]):
                    for step in range(TRAINING_CONFIG["steps_per_generation"]):
                        financial_data = torch.randn(
                            1, HNN_LUT_CONFIG["timesteps"], HNN_LUT_CONFIG["input_features"], 
                            dtype=SYSTEM_CONFIG["precision_dtype"]
                        ).to(device)
                        
                        target_action_label = torch.tensor([random.randint(0, ND_AI_HEAD_CONFIG["output_actions"] - 1)], device=device)

                        optimizer.zero_grad()
                        action_str, ai_confidence, grok_confidence, logits = current_system(financial_data)
                        
                        loss = criterion(logits, target_action_label)
                        loss.backward()
                        optimizer.step()

                        simulated_roi = env_simulator.simulate_trade_outcome(action_str, ai_confidence)
                        
                        current_individual_roi.append(simulated_roi)
                        current_individual_ai_conf.append(ai_confidence)
                        current_individual_grok_conf.append(grok_confidence)
                        
                        if device == "cuda":
                            current_vram_usage_gb = torch.cuda.memory_allocated() / (1024**3)
                            if current_vram_usage_gb > GA_CONFIG["vram_limit_gb"]:
                                logger.warning(f"  Individualas {individual_idx+1}: VRAM limitas viršytas ({current_vram_usage_gb:.2f}GB). Nutraukiama.")
                                raise RuntimeError("VRAM Limit Exceeded") 
                
                if device == "cuda":
                    current_vram_usage_gb = torch.cuda.memory_allocated() / (1024**3)
                else:
                    current_vram_usage_gb = 0.0 
                                    
                avg_roi = np.mean(current_individual_roi) if current_individual_roi else 0.0
                avg_ai_conf = np.mean(current_individual_ai_conf) if current_individual_ai_conf else 0.0
                avg_grok_conf = np.mean(current_individual_grok_conf) if current_individual_grok_conf else 0.0

                current_generation_feedback.append({
                    "roi": avg_roi,
                    "ai_confidence": avg_ai_conf,
                    "grok_confidence": avg_grok_conf,
                    "vram_used_gb": current_vram_usage_gb
                })
                
                logger.info(
                    f"  Individualas {individual_idx+1} [OK]: Parametrai={individual['params']}, "
                    f"Vid. ROI={avg_roi:.4f}, Vid. Patikimumas={avg_ai_conf:.2f}, VRAM={current_vram_usage_gb:.2f}GB"
                )
                
            except RuntimeError as e:
                logger.error(f"  Individualas {individual_idx+1} [KLAIDA]: {e}. Parametrai: {individual['params']}")
                current_generation_feedback.append({
                    "roi": -1.0, 
                    "ai_confidence": 0.0,
                    "grok_confidence": 0.0,
                    "vram_used_gb": float('inf') 
                })
            finally:
                del current_system 
                clean_gpu_memory() 

        ga_orchestrator.evolve_population(current_generation_feedback)
        
        best_params_this_gen = ga_orchestrator.get_best_individual_params()
        
        best_model_this_gen = UnifiedHNN_AIFabric(best_params_this_gen).to(device)
        best_model_this_gen.eval() 

        test_roi = []
        with torch.no_grad():
            for _ in range(TRAINING_CONFIG["steps_per_generation"]):
                data = torch.randn(
                    1, HNN_LUT_CONFIG["timesteps"], HNN_LUT_CONFIG["input_features"], 
                    dtype=SYSTEM_CONFIG["precision_dtype"]
                ).to(device)
                action, conf, _, _ = best_model_this_gen(data) 
                test_roi.append(env_simulator.simulate_trade_outcome(action, conf))
        
        avg_gen_roi = np.mean(test_roi)
        total_system_roi_history.append(avg_gen_roi)
        
        logger.info(f"--- Karta {generation+1} Pabaiga: Geriausi parametrai={best_params_this_gen}, Vid. testavimo ROI={avg_gen_roi:.4f} ---")
        
        del best_model_this_gen
        clean_gpu_memory()


    logger.info("\n--- Evoliucija baigta ---")
    logger.info(f"Sistemos ROI evoliucija per kartas: {total_system_roi_history}")
    
    final_best_params = ga_orchestrator.get_best_individual_params()
    final_system = UnifiedHNN_AIFabric(final_best_params).to(device)
    
    logger.info(f"Galutinio modelio parametrai: {final_best_params}")

if __name__ == "__main__":
    main()
```
